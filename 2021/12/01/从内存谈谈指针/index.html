  • 2021-12-01

#引子!
只要是C/C++的书籍都会谈到“指针储存数值地址”我们来深挖一下这句话
首先是数值地址：
与python不同，在c++中，使用任何变量都必须先下定义：如int a,double b等等，这样做的目的实际上是让计算机分配一块地址给对应的变量从而方便我们操作
说起地址我们就要谈谈储存单元，单元最小的是bit,它只有两种状态,0和1（也可以是高压和低压），八个bit组成一个byte。
byte就很重要了，他是编码一个字符最小的可寻址单元.
维基百科对byte（字节）的定义如下通常用作计算机信息计量单位，不分数据类型。[1][2] 是通信和数据存储的概念。一个字节代表八个比特。从历史的观点上，“字节”表示用于编码单个字符所需要的比特数量。历史上字节长度曾基于硬件为1-48比特不等，最初通常使用6比特或9比特为一字节。今日标准以8比特作为一字节，因8为二进制整数。
从ASCII表中可以看到，一个byte储存的数字可以映射对应的符号,这是计算机与人交流的基础（映射），在ASCII表中：数字 32–126 分配给了能在键盘上找到的字符,当你查看或打印文档时就会出现。注:十进制32代表空格 ,十进制数字 127 代表 DELETE 命令。0–31 分配给了控制字符，用于控制打印机等外围设备（比如计算机视觉中的指令：cv2.videocapture(0)中的0就代表摄像头，127-255属于扩展ASCII打印字符，他们包括了一些数学符号,汉字等非英语字母的符号
换算：一个byte储存256个bit,1024个byte组成kb,1024个kb组成mb…这样循环下去.可以从这里直观理解到内存的概念
而电脑常说的32位，64位（现在我们一般用64位）是什么意思呢？这是说在计算机架构中，64位整数、内存地址或其他数据单元，它们最高达到64位（8字节）宽，这样在处理多媒体内容（指文本，图像，声音混合的媒体）时有更加的性能。

#回到指针这个话题!
在定义一个变量时，计算机为其在茫茫大海中分了一小块地方，成了它的私人空间，但作为操作者的我们如何准确找到它？这就是指针的意义所在了
指针怎么用？定义+赋值+取值
定义：我们要怎样区别指针和其他变量来告诉计算机：”这是个指针！”呢？简单，直接在指针名前面加上一个特殊的* 如int a
赋值：通过符号&来完成： a=&b ,这样就把b的地址赋给了a
取值：现在的a代表b的地址，我们又可以通过符号来把b的地址中存在的b给取出来,如*a

有人可能会说:你是猪吗？我直接不要a，想要b的地址就写&b,想要b的值就直接写b不就行了吗？
我的回答是:你错了，从一个例子来看一下：
定义这样一个函数:
void swap(a,b)
{
int tempt;
tempt=a;
a=b;

看似一顿操作猛如虎，其实我们将它放入主函数中
int main(){
int x=1;
int y=2;
swap(x,y);
cout<<x<<” “<<y<<”\n”;

}
发现输出为2,1;1,2
这是为什么呢？这就涉及到了形参与实参的概念：
形参(parameter)： 全称为”形式参数” 由于它不是实际存在变量，所以又称虚拟变量。 是在定义函数名和函数体的时候使用的参数,目的是用来接收调用该函数时传入的参数. 在调用函数时，实参将赋值给形参。
在上面的例子中，在main()里面，swap()函数中的形参a,b直接把x,y的值拷贝了过来并进行了交换，但是对于x,y并没有实际的影响，这就是不用指针的下场！

现在我们改用指针来做：
void swap(*a,*b)
{
int tempt;
tempt=*a
*a=*b;
*b=tempt;
cout<<*a<<” “<<*b<<”\n”;
int main(){
int x=1;
int y=2;
swap(&x,&y); //注意，这里把x,y的地址赋给形参，而不是把x,y赋给形参
cout<<x<<” “<<y<<”\n”;

}
输出结果：2，1；2，1
